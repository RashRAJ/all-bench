## [0.2.0] - 2026-07-24

### Added
- `update` command — downloads and installs the latest GitHub release of `all-bench` in place, matching your OS/arch automatically
- aiperf custom dataset support: `input_file` / `custom_dataset_type` config fields (and matching `--input-file` / `--custom-dataset-type` flags) to replay a dataset instead of synthesizing prompts

### Changed
- Removed dead/commented-out legacy implementation left over in `cmd/diff.go`

## [0.1.1] - 2026-05-07

### Added
- `diff` command — compare the last two benchmark runs side-by-side
- Benchmark history saved to `~/.all-bench/history/` as timestamped JSON snapshots
- Config diff section highlighting changed fields between runs
- Verdict banner showing overall improvement/regression with metric count
- Key metrics cards (TTFT, Req Latency, Throughput, Duration) with colored change indicators
- Detailed comparison table (Run A → Run B) with per-metric spark bars and percentage change
- Percentile distribution table (avg, p50, p90, p99, min, max) with per-cell coloring
- Lipgloss-powered styling with rounded borders, truecolor palette, and run info header

## [0.1.0] - 2026-04-29

### Added
- `run` command — invoke aiperf or vlmbench via a single YAML config
- `list` command — show which runners are installed
- Normalized table and JSON output across runners
- `--runner`, `--format`, and `--out` flags
- GoReleaser packaging for macOS, Linux, and Windows
